# Mini-Projeto Avaliativo: Análise Exploratória de Dados (Varejo)
**Turma:** Analise_de_Dados_T1  
**Estudante:** Filipe de Oliveira Gomes  

Este repositório contém a solução do mini-projeto de Análise Exploratória de Dados (AED) aplicada a uma base de dados de varejo, com o objetivo de realizar o tratamento, limpeza e extração de insights estatísticos utilizando a biblioteca `pandas`.

---

## 🚀 Como Executar o Projeto

### Pré-requisitos
Certifique-se de ter o Python e a biblioteca `pandas` instalados na sua máquina. Caso não tenha o pandas, instale via terminal:
```bash
pip install pandas
